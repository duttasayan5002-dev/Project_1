# Personal Information Manager
Project Description:
  This is a simple Python project that stores and displays personal information.  
  It also takes user input for additional details and shows a formatted output.
Features
- Stores static personal information
- Takes user input from the user
- Validates empty input
- Converts age into months
- Displays formatted output
## Concepts Used
- Variables
- Input and Output
- Conditional statements (if-else)
- String methods (strip, title)
- Arithmetic operations
- String formatting (f-strings)
- ## How to Run
Run the Python file using:

python personal_info.py
